popup(blocking=True, parent=None)[source]¶

Create and show the menu as context or popup menu.

If the menu has not been initialized when this method is called, it will be setup and initialized automatically.

Parameters:
  • blocking (boolean, optional) – Whether or not the popup menu should be displayed in a blocking context. If False, then this menu object must be stored on the heap or it will be immediately garbage collected. Defaults to True. This option may not be supported on all toolkit backends.
  • parent (Toolkit widget or None, optional) – The parent on which to popup the menu. This only has an effect on menus which are created outside of the normal Enaml heirarchy and are setup and initialized during the call to this method. Even then, the popup menu will be a top-level widget, but it’s lifetime will be tied to that of the parent.
