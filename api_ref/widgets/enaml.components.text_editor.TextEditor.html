TextEditor¶

class enaml.components.text_editor.TextEditor[source]¶

Bases: enaml.components.control.Control

A text editor widget capable of editing plain text with styles.

This is not a general text edit widget with general capabilties for sophistcated formatting or image display.

read_only = None¶

Whether or not the editor is read only.

wrap_lines = None¶

Whether to wrap text in the editor or not. If True, keep words together if possible.

overwrite = None¶

Whether to overwrite or insert text when the user types.

cursor_position = None¶

The position of the cursor in the editor.

anchor_position = None¶

The anchor of the selection in the editor.

cursor_line = None¶

A read only property holding the line number of cursor in the editor.

cursor_column = None¶

A read only property holding the column number of cursor in the editor.

modified = None¶

A read only property that is set to True if the user has changed the line edit from the ui, False otherwise. This is reset to False if the text is programmatically changed.

selected_text = None¶

A read only property that is updated with the text selected in the editor.

text_changed = None¶

Fired when the text is changed programmatically, or by the user via the ui. The event does not carry a payload. To retrieve the current text, use the get_text() method.

text_edited = None¶

Fired when the text is changed by the user explicitly through the ui and not programmatically. The event does not carry a payload. To retrieve the current text, call get_text().

lost_focus = None¶

Fired when the widget has lost input focus.

hug_width = None¶

How strongly a component hugs it’s contents’ width. TextEditors ignore the width hug by default, so they expand freely in width.

hug_height = None¶

How strongly a component hugs it’s contents’ height. TextEditors ignore the height hug by default, so they expand freely in height.

abstract_obj = None¶
