DateEdit¶

class enaml.components.date_edit.DateEdit[source]¶

Bases: enaml.components.bounded_date.BoundedDate

A widget to edit a Python datetime.date object.

A DateEdit displays a Python datetime.date using an appropriate toolkit specific control. This is a geometrically smaller control than what is provided by Calendar.

date_format = None¶

A python date format string to format the date. If none is supplied (or is invalid) the system locale setting is used. This may not be supported by all backends.

date_changed = None¶

Triggered whenever the user changes the date through the ui control, but not programmatically. The event payload will be the date on the control.

hug_width = None¶

How strongly a component hugs it’s contents’ width. DateEdits ignore the width hug by default, so they expand freely in width.

Overridden parent class trait.
