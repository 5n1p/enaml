enaml.core¶

The enaml.core package contains all of the core functionality of the Enaml toolkit.

toolkit Module¶

class enaml.core.toolkit.Toolkit[source]¶

Bases: dict

The Enaml Toolkit class which facilitates toolkit independent development.

The Toolkit is a dict subclass which is injected between the global and builtin scopes of an executing an Enaml function or expression. The contents of the dictionary may consist of any useful objects that should be automatically in scope for executing Enaml code. Typical content of a toolkit will include component constructors, layout helpers, operators, and useful toolkit abstractions.

class enaml.core.expressions.ExecutionScope(obj, identifiers, f_globals, toolkit, overrides, cb)[source]¶

Bases: object

A custom mapping object that implements the scope resolution order for the evaluation of code objects in Enaml expressions.

Note

Strong references are kept to all objects passed to the constructor, so these scope objects should be created as needed and discarded in order to avoid issues with reference cycles.
