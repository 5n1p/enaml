MdiWindow¶

class enaml.widgets.mdi_window.MdiWindow(parent=None, **kwargs)[source]¶

Bases: enaml.widgets.widget_component.WidgetComponent

A widget which can be used as a window in an MdiArea.

An MdiWindow is a widget which can be used as an independent window in an MdiArea. It can have at most a single child widget which is an instance of WidgetComponent.
