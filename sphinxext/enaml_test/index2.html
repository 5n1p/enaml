Radio Group Std Library¶

A custom RadioGroup component.

A RadioGroup makes it easier to work with a group of radio buttons since it manages the various toggle behavior internally and yields output in the form of selected item, index, and label.

Enaml declared component enaml.stdlib.radio_group.RadioGroup(Container)¶

A Container that automatically generates a group of RadioButtons.

orientation : string
The orientation of the group of radio buttons, either ‘horizontal’ or ‘vertical’. Defaults to ‘horizontal’.
items : list
The list of items which represent the underlying radio buttons.
to_string : callable
A callable which get mapped across the list of items to generate the list of labels for the buttons.
selected : object
An attribute which is updated with the item associated with the currently selected radio button. Changing this value will cause a corresponding change the appropriate button.
selected_index : integer
The integer index of the currently selected radio button. This attribute is updated after the ‘selected’ attribute is updated.
selected_label : string
The label of the currently selected radio button. This attribute is updated after the ‘selected_index’ attribute is updated.
