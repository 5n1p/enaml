class enaml.tests.components.operators.TestLessLess(methodName='runTest')[source]¶

Bases: enaml.tests.components.enaml_test_case.EnamlTestCase

Generic testcases for enaml the LessLess operator.

These testcases test the basic functionality of widgets interacting with each other.

To run these tests for a specific back-end it is required to have a functional implementation of the following wigdets:

  • Window
  • CheckBox
  • Label
