TestGroupBox¶

class enaml.tests.components.group_box.TestGroupBox(methodName='runTest')[source]¶

Bases: enaml.tests.components.enaml_test_case.EnamlTestCase

Logic for testing labels.

Tooklit testcases need to provide the following methods:

get_title
Returns the group title from the tookit widget
get_flat
Returns the flat border style from the tookit widget
get_title_align
Returns the title align style from the tookit widget

Note

All the provided methods need to support the extented signature <method>(component, widget).
