Control¶

class enaml.widgets.control.Control[source]¶

Bases: enaml.widgets.component.Component

The base class of all leaf widgets in Enaml. Controls cannot have children and attempts to add children to a control will raise an exception.

error = None¶

A boolean which indicates whether an exception was raised through user interaction or setting a value trait on the Control.

exception = None¶

The exception object that was raised to create the error state.

Overridden parent class trait

capture_exceptions()[source]¶

capture_notification_exceptions(handler=None, reraise_exceptions=True, main=False, locked=False)[source]¶

Return a ShellNotificationContext that will set error state automatically, including TraitErrors in listeners.
