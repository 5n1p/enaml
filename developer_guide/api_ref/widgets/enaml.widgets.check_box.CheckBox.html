CheckBox¶

class enaml.widgets.check_box.CheckBox[source]¶

Bases: enaml.widgets.toggle_control.ToggleControl

A check box widget.

Use a check box when it’s necessary to toggle a boolean value independent of any other widgets in the group. For allowing the toggling of only one value in a group of values, use a group of radio buttons.
