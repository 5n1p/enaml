Welcome to Enaml’s documentation!¶

Enaml is Not A Markup Language. Enaml is a tool for building user interfaces. It is a declarative language based on Python, analogous to Qt’s QML. Enaml currently supports wxWidgets as a backend via wxPython, and Qt as a backend via PySide and PyQt4. However, it is not tied to a single widget toolkit.

Warning

Enaml is currently under heavy development and the documentation is not always up-to-date. Please recompile in order to have the latest changes from the source code.
