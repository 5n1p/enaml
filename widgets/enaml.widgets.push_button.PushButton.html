Push Button¶

class enaml.widgets.push_button.PushButton[source]¶

Bases: enaml.widgets.control.Control

A push button widget.

down = None¶

A read only property which indicates whether or not the button is currently pressed.

text = None¶

The text to use as the button’s label.

clicked = None¶

Fired when the button is pressed and released.

pressed = None¶

Fired when the button is pressed.

released = None¶

Fired when the button is released.

hug_width = None¶

How strongly a component hugs it’s contents’ width. PushButtons hug their contents’ width weakly.

abstract_obj = None¶

Overridden parent class trait

Sample¶

System Wx Qt Enaml
Windows 
defn MainWindow:
    Window:
        title = "Push button example"
        constraints = [vertical(top, button, bottom),
                        horizontal(left, button, right)]
        PushButton -> button:
            text = "Press me"
