Control¶

class enaml.widgets.control.Control[source]¶

Bases: enaml.widgets.component.Component

The base class of all leaf widgets in Enaml. Controls cannot have children and attempts to add children to a control will raise an exception.

A boolean which indicates whether an exception was raised through user interaction or setting a value trait on the Control.
